# BeOCR-Business-Central-Conector
Conecta tu Business Central con BeOCR para sacarle mayor partido a la herramienta, introduce tickets, factura de proveedores y mucho mas.

Pasos para conectar tu Business Central a BeOCR

1. Instalar la app de este proyecto.
  1.1. Vaya al buscador y seleccione la primera opción que aparece buscando "Administración de Extensiones".
  1.2. Depliegue el menú "Administrar" y pulse sobre "Cargar extensión..."
  1.3. Carge el .app de este repositorio y acepte las condiciones.
2. Dar de alta en Be OCR.
   2.1. Vaya a la url https://beocr.com.
   2.2. Registrese y valide su cuenta de correo.
   2.3. Acceda a la web y vaya al menú "APIKey".
   2.4. Crea una API Key y guardela en un lugar seguro para luego utilizarla.
3. Configurar Conector de Be OCR.
   3.1. Vaya al buscador y seleccone la primera opción que aparece buscando "BeOCR - Configuración".
   3.2. Cree un nuevo registro.
   3.3. La URL se cargará por defecto pero tiene que indicar la API Key generada en el campo "API Key".
   3.4. Es necesario generar un nuevo Nº de Serie para los documentos de Be OCR y configurarlo en "Nº de Serie de documentos OCR".
4. Usar el conector.
   4.1. Vaya al buscador y seleccione la primera opción que aparece buscando "BeOCR - Lista de documentos".
   4.2. Pulse en "Cargar Documento BeOCR" para subir una Factura o ticket de compra a Be OCR.
   4.3. Después de haber procesado, cuando le pulse "Sicronizar BeOCR" podra descargar los datos de la factura.
